게임 방법

  1. 사진을 보고 질문을 만들어요. (총 10문제!)
  2. 질문을 등록하면 **바로 점수와 피드백**이 나와요.
  3. [힌트] 버튼을 누르면 좋은 질문 팁을 볼 수 있어요.
  4. 좋은 질문은 점수가 높아요! (최대 1000점)
    • 👍 **높은 점수:**
      • "만약~?", "왜~?", "나라면~?" 같은 깊은 생각 질문
      • 나의 경험이나 생각을 **길게(50자 이상)** 쓴 성의 있는 질문
    • 👎 **낮은 점수:**
      • "누가?", "몇 개?" 같은 단순 관찰 질문
      • "이게 뭐야?" 처럼 **너무 짧은(7자 미만)** 질문
  5. 이런 질문은 0점이에요! (수정 기회 1번 줌)
    • 욕설, 비속어, 성의 없는 표현 (예: ㅋㅋ, 123, 몰루?)
    • 오탈자 (예: 않되, 어떻해)
    • 질문 형식이 아닌 말 (예: ~까?, ~요? 로 끝나지 않음)
